# legacy/

Prototipo visual anterior do Friends. Mantido como referencia historica ate
que as pecas uteis sejam portadas para a nova estrutura `frontend/` + `backend/`.

Arquivos:

- `friends-dashboard.jsx`: SPA monolitica com dados em memoria, parsers de CSV e
  VCF, paleta de cores, tags sugeridas e logica de temperatura. Base de consulta
  para a atividade de porting descrita em 13.1 do PRD.

Sera removido quando o frontend real cobrir o equivalente funcional.
